methodology

For each kernel series ≥ 6.12 the latest patch release is resolved from https://cdn.kernel.org/pub/linux/kernel, the source tarball downloaded and only *.rs files extracted. File sizes come from the extracted sources; line counts (code / comment / blank) from 1.98. Generated bindings that are produced only at build time are not shipped in the tarball and therefore not counted. Data regenerated with scripts/build.sh; machine-readable results live in data/kernels.json.
